### 3.1 Workspace.html（工作台）

> **文件位置**：`pages/workspace.html`
> **purpose**：数字员工工作台主界面，支持标准视图/老板视图切换，整合对话和看板功能
> **current_state**：包含视频背景、履约对象入口、AI员工卡片、主视图切换（看板/对话）
> **constraints**：深色科技风、此文件不走L2直接全页跳转
> **关联文件**：css/workspace.css、css/layout.css、assets/videos/office.mp4

---

#### 原始需求（用户原话）

> "数字员工工作台要像一个指挥中心，左边是数字员工，右边是任务。"

3. **老板视图**：沉浸式视频背景（office.mp4），隐藏所有业务元素

4. **履约对象入口**：点击进入履约中心（performance.html）

#### 关键代码片段

**1. 视图切换（标准 ↔ 老板）**：
```javascript
function toggleView(viewType) {
  var grid = document.getElementById('aiGrid');
  var video = document.getElementById('bossVideo');
  var bizDesc = document.querySelector('.biz-desc');
  var aiTabs = document.querySelector('.ai-tabs');
  
  if (viewType === 'boss') {
    grid.classList.add('boss-view');
    if (bizDesc) bizDesc.style.display = 'none';
    if (aiTabs) aiTabs.style.display = 'none';
    video.play();
  } else {
    grid.classList.remove('boss-view');
    if (bizDesc) bizDesc.style.display = 'block';
    if (aiTabs) aiTabs.style.display = 'flex';
    video.pause();
  }
}
```

**2. 履约入口跳转**：
```html
<!-- 履约对象卡片点击 -->

<!-- 右侧sidebar点击 -->

```

**3. 视频元素**：
```html
<video id="bossVideo" class="boss-bg-video" style="display:none;" 
       preload="auto" muted loop playsinline>
  <source src="../assets/videos/office.mp4" type="video/mp4">
</video>
```

#### 约束条件

| 条件 | 要求 |
|------|------|
